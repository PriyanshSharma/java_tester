Problem statement -- 


I have a car. 
There are some features of the class --> drivecapability, luxurycapability, fastSpeedCar capability.
There are some types for the CAR --
1. Passenger Car - 
	normal Drive capability
	normal luxury
	normal speed car
	
2. Racing car --
	- Special drive capability 
	- A very good luxury
	- Very fast engine for fastDrive capability
	
3. LuxuryCar --
	- Special drive capability
	- A high class luxury
	- A very fast engine for fast drive capability - Same as Racing Car


Now in future there can be different cars can come up, with some or the features to be same or to be different.

Design LLD for the same.
